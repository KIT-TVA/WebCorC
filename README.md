# WebCorC Wiki
Configure the wiki by editing `mkdocs.yml`. All wiki files are located in `docs`.

## Building
Navigate to the root folder and run
```sh
mkdocs serve
```

## Deploying
Navigate to the root folder and execute
```sh
mkdocs gh-deploy
```
The page will be available at https://kit-tva.github.io/WebCorC/. The generated files are hosted in the `gh-pages` branch.
